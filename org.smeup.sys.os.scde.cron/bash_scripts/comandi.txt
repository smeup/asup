- Visualizza log cron tab:   grep CRON /var/log/syslog

Chiamata script per registrazione crontask:

*** addcron ***

	Parametri:	1) Maschera cron
				2) Server
				3) Id task
				4) Utente
				5) Comando (tra doppi apici)
				6) Nome task				
				7) Descrizione task (tra doppi apici)
	Esempio: ./addcron "* * * * *" 127.0.0.1 123456 FORDAR "WRKOBJ OBJTYPE(*PGM) OBJ(P_MULT/*ALL)" WRKOBJ_PGM "Descrizione del task"

*** removecron ***

	Parametri: 1) Nome task o Id task 	
 	
 	Esempio:  ./removecron WRKOBJ_PGM
